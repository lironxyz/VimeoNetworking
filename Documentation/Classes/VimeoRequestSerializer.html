VimeoRequestSerializer

final public class VimeoRequestSerializer

VimeoRequestSerializer handles request serialization as well as adding Vimeo-specific authorization headers to outbound requests. It can be initialized with either a dynamic AccessTokenProvider or a static AppConfiguration.
