            retryPolicy: RetryPolicy? = nil)

    Parameters

    method

    the HTTP method (e.g. .GET, .POST), defaults to .GET
    path

    url path for this request
    parameters

    additional parameters for this request
    modelKeyPath

    optionally query a nested JSON key path for the response model object to be returned, defaults to nil
    cacheFetchPolicy

    describes how this request should query for cached responses, defaults to .CacheThenNetwork
    shouldCacheResponse

    whether the response should be stored in cache, defaults to true
    retryPolicy

    describes how the request should handle retrying after failure, defaults to .SingleAttempt

    Return Value

    an initialized Request
